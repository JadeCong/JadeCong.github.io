# Human Pose Estimation

> 1. **Human3.6M**: [Large-scale 3d human motion capture datasets for fitness, close human interactions, and self-contact](http://vision.imar.ro/human3.6m/description.php "2018-Romanian National Authority for Scientific Research") [Human3.6M, Human Motion]
> 2. **MS COCO**: [A large-scale object detection, segmentation, and captioning dataset](https://cocodataset.org/#home "2015-Microsoft") [Common Objects, Context]
> 3. **MPII**: [2D Human Pose Estimation - New Benchmark and State of the Art Analysis](http://human-pose.mpi-inf.mpg.de "2014_06-Max Planck Institute for Informatics-Mykhaylo Andriluka") [Human Pose Estimation, Benchmark]
> 4. **CrowdPose**: [Efficient Crowded Scenes Pose Estimation and A New Benchmark](https://drive.google.com/file/d/1VprytECcLtU4tKP32SYi_7oDRbw7yUTL/view "2019_01_23-Shanghai Jiao Tong University-Jiefeng Li") [Pose Estimation, Crowded Scene]

# Object Detection

> 1. **Objects365**: [A Large-scale, High-quality Dataset for Object Detection](https://www.objects365.org/overview.html "2024_08-Megvii Technology-Shuai Shao") [Object detection, Benchmark]
> 2. **Open Images Dataset**: [Unified image classification, object detection, and visual relationship detection at scale](https://storage.googleapis.com/openimages/web/index.html "2022_10-Google AI") [Image classification, Object detection]

# Large-scale Multi-modality Model

> 1. **RefCOCO**: [Large-scale Multi-modality Models Evaluation Suite](https://huggingface.co/datasets/lmms-lab/RefCOCO "2014_10-Northeastern University") [Multi-modality Models, Evaluation]
> 2. **Molmo**: [Multimodal Open Language Model](https://github.com/allenai/molmo "2024_12_05-Allen AI") [Multimodal Language Model]
> 3. **Data Commons**: [A single source for publicly available statistical data](https://datacommons.org/ "2026-Google") [knowledge graph, schema, APIs, open-source project platform]
> 4. **Wikidata Vector Database**: [semantic embeddings of Wikidata entities](https://wd-vectordb.wmcloud.org/ "2025_10_01-Wikimedia") [Wikidata, Vector Database, semantic search, knowledge graph]
> 5. **LAION Science Dataset Explorer**: [Interactive visualization and exploration of scientific papers from the Aella open science dataset](https://laion.inference.net/embeddings "2025-LAION") [scientific papers, semantic embeddings, dimensionality reduction, clustering visualizations]

# Physics Simulation

> 1. **The Well**: [15TB of Physics Simulations](https://polymathic-ai.org/the_well/ "2024-Polymathic AI") [Physics Simulations, machine learning datasets]
